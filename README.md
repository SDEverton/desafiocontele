## Iniciando

Primeiro devemos instalar as dependências e adicionar as variáveis de ambiente:

Arquivo: next.config.js
```
module.exports = {
  env: {
    mapsKey: 'API_KEY_MAPS',
  },
}

```

```bash
npm install
#or
yarn

npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) para ver o resultado.
